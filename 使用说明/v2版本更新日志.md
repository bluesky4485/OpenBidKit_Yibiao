# 易标投标工具箱版本更新日志

## v2.18.2

### 优化

- 小尺寸 Markdown 卡片不再显示不适用的“全屏”入口，界面更加简洁。

## v2.18.1

### 新增

- 新增 HTML 图表生成能力，可按需生成甘特图、进度网络图、组织架构图、泳道图、RACI 职责矩阵、风险矩阵、系统架构图、WBS、鱼骨图及常用统计图表。
- 新增 HTML 图表类型和生成数量配置，可根据标书内容控制配图范围。
- 新增独立的图片编排阶段，在正文完成后统一规划 AI 配图、Mermaid 图和 HTML 图表。

### 优化

- 支持使用多个连续小节共同生成一张综合图表，并根据图表用途自动选择插入位置。
- 优化配图优先级和冲突处理，同一小节只保留最合适的一张图片。
- 收窄 Mermaid 图表范围，重点用于流程、层级和职责关系表达，降低复杂图表生成失败的概率。
- 强化图片差异化约束，减少相似图片和重复配图。

### 修复

- 修复正文配图可能重复的问题。
- 修复 AI 接口返回成功但没有有效内容时，任务仍被误判为成功的问题。

### 调整

- 移除龙猫平台支持，请改用其他受支持的 AI 服务或 OpenAI 兼容接口。
